Design System · NavBar

Responsive Header
Navigation Component

Sticky scroll · Dropdowns · Section sidebar · Mobile section panel · Light / Dark / Glass

Overview

This section uses data-title="Overview". When the JS detects any <section data-title> elements on the page it automatically builds a fixed left-side pill menu on desktop and adds an "On this page" panel to the mobile drawer — no manual wiring needed.

Features

Scroll down to watch the active indicator track which section is currently in view. The dot snaps to the correct item using an IntersectionObserver with no external dependencies.

Installation

Copy the <header>, the .ds-section-nav element, all CSS between the DS markers, and the JS IIFE into your page. The JS autodiscovers section[data-title] elements at runtime.

Configuration

Override the CSS custom properties on :root or any ancestor to theme the component. Switch between light, dark, and glass by setting data-variant on the <header>. The sidebar mirrors the same attribute automatically.

Accessibility

All interactive elements carry correct ARIA roles, states, and labels. The section sidebar is a <nav> landmark distinct from the header nav. Dropdowns close on outside click and Escape.

Changelog

v2 adds the section nav sidebar and mobile section drawer panel, driven by IntersectionObserver. v1 introduced the base navbar with dropdowns, animated hamburger, and three colour variants.
